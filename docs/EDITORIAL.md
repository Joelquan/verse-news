# Verse News — Editorial Standards

**Brand spine:** Verse News reports the biblical story as news; Creation helps readers understand the world that story is set in; Life helps them walk it on Monday.

If a feature does not serve that sentence, it is a **toolbox item** (Bible reader, games, maps) — valuable, but below the fold, not the identity.

---

## House rules (all desks)

| Rule | Standard |
|------|----------|
| **Form** | Headline + summary + body + secondary panel + voices. Same skeleton, different labels. |
| **Never pretend** | Editorial paraphrase ≠ quotation. Scripture always labeled as Scripture. Science never labeled as Scripture. |
| **Sources visible** | Every piece names methods, missions, passages, or classic writers — not vague “scientists say.” |
| **Tone** | Clear, adult, non-mocking. No culture-war clickbait; no sneering at faith or science. |
| **Length** | Earn the scroll. Short is fine for Life; Creation and major Bible reports should be substantial. |
| **Art** | Image supports the claim; no random mismatches. Prefer none over wrong. |

---

## Desk A — Bible / Scripture reports (masthead)

| | |
|--|--|
| **Job** | Report a biblical event or teaching moment as if filing from the scene — then hand the reader the text. |
| **Primary reader** | Anyone who wants the sweep of the story, not only a verse. |
| **Tabs** | **NEWS** · **SCRIPTURE** · **COMMENTS** (historic voices) |
| **Body is** | Narrative reportage: stakes, people, place, turning point. Not a sermon outline. |
| **Scripture panel is** | Actual verse text + ref. Never “Finding / Method.” |
| **Commentary is** | Historic Christian writers clearly *not* equal to the text. |
| **Anchor verse** | Required when possible. |
| **Success** | Reader finishes the report and opens the passage. |
| **Failure modes** | Headlines that oversell; empty scripture arrays; “news” that is really a devotion. |
| **Bar** | Prefer fewer complete stories over hundreds of thin ones. Gospels / core arcs > long-tail stubs. |

---

## Desk B — Creation (science of the made world)

| | |
|--|--|
| **Job** | Established science that invites appreciation of creation — not a debate club, not a textbook dump. |
| **Primary reader** | Curious believers, students, science-friendly skeptics who will check claims. |
| **Tabs** | **REPORT** · **KEY FACTS** · **VOICES** |
| **Body is** | Long-form science feature: methods, numbers, history of discovery, cross-checks, open questions. |
| **Key Facts is** | Bullet findings (Finding, Evidence, Method…). Not KJV. |
| **Voices is** | Mission teams, papers, classic scientists — attributed. Optional light creation-appreciation in body only. |
| **Anchor verse** | Optional short frame (e.g. Gen 1:1, Ps 19) — **frame, not proof of a lab result.** |
| **Success** | Reader learns something real *and* feels wonder, without feeling baited. |
| **Failure modes** | Thin “wow” paragraphs; overclaiming theology from data; fake precision; same voice as Bible “news.” |
| **Bar** | Cite real missions/experiments; separate **fact / interpretation / appreciation**. Prefer peer-consensus science. |

---

## Desk C — Life (everyday Christian guidance)

| | |
|--|--|
| **Job** | Pastoral, practical help for niches of real life — marriage, anxiety, money, work — rooted in Scripture. |
| **Primary reader** | Someone searching a *situation*, not a biblical battle report. |
| **Tabs** | **GUIDE** · **SCRIPTURE** · **VOICES** |
| **Body is** | Short–medium counsel: honest, specific, non-shaming, actionable. |
| **Scripture is** | Passages that ground the counsel — few, well chosen. |
| **Voices is** | Optional classic pastoral wisdom — never influencer fluff. |
| **Success** | Reader feels seen and has a next step. |
| **Failure modes** | Generic self-help with a verse sticker; culture-war rant; medical/legal claims you cannot support. |
| **Bar** | One issue per piece; clear limits (“not a therapist / not a lawyer”); warmth over cleverness. |

---

## What each desk is **not**

| Desk | Not |
|------|-----|
| **Bible** | Not a systematic theology textbook; not Creation apologetics first. |
| **Creation** | Not “Bible answers to every controversy”; not anti-science; not worship lyrics without data. |
| **Life** | Not news reporting of ancient events; not a science feature; not a replacement for church/counseling. |

## Homepage / first-minute standards

1. **Masthead job:** Lead with a **Bible report** unless Creation is deliberately scheduled.
2. **One-liner under brand:** Biblical events as news — plus Creation science & Life guides.
3. **No product-lab chrome on first visit:** Classic/New home switch lives in the footer, not the hero.
4. **Don’t surface empty desks** until they are stocked.
5. **Toolbox** (Bible, games, maps, weather) supports reading — it does not replace the paper.

---

## Weekly editorial check

1. **Masthead:** Did home lead with a Bible report unless we deliberately scheduled Creation?
2. **Honesty:** Any science labeled as scripture? Any paraphrase labeled as KJV?
3. **Depth:** Would a careful outsider call this thin?
4. **Desk fit:** Could this piece only live on *one* desk? If it could live on all three, rewrite until it can’t.
5. **Return visit:** Is there a reason to open the site *tomorrow* (daily story, VOTD, new guide)?
